# Brownies

<!-- more -->

## Zutaten

* 250g Butter
* 225g Zucker
* 150g brauner Zucker
* 1/2 Päckchen Vanillezucker
* 100g Kakao
* 3 Eier
* 200g Mehl
* 1/2 Päckchen
* 200g Nüsse oder weiße Schokolade

## Zubereitung

* Butter in einem großen Topf schmelzen lassen, dann restliche Zutaten hinzugeben
* für 30-35 Minuten bei 180°C backen
